# Giovany Kantoro Portfolio

Personal portfolio website for Giovany Kantoro, built with React and Vite.

## Development

```bash
pnpm install
pnpm dev
```

## Production Build

```bash
pnpm build
```

The production output is generated in `dist/`.

## Vercel Settings

- Framework Preset: Vite
- Install Command: `pnpm install`
- Build Command: `pnpm build`
- Output Directory: `dist`
